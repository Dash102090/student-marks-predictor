# Student-Marks-Predictor

## 💡About:
This program illustrates how scikit works with pandas, tkinter and matplotlib.

## 📦 Requirement:

Open your terminal by opening start and typing *`cmd`*, enter this command line in your terminal **`pip install sklearn matplotlib`**

## 🟢Pros:

1. Can understand how scikit learn's linear model works with `Pandas` dataframe and `Matplotlib`.
2. Can put your own file and set X and y to predict what you want.
3. Shows you the data of your file.
4. Shows Visual representation of your prediction

## 🔴Cons:

1. Requires python knowledge for changing the prediction.
2. Only predicts with no additional uses.
3. Messy code.
